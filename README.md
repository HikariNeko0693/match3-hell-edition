# 武龚郝高消消乐（地狱版）

一个用 **Phaser 3** 写的网页版三消小游戏，支持 PC 和 手机浏览器游玩。  
本项目受启发于小红书@呵呜啊哗的二创视频，指路：http://xhslink.com/o/6m6nS5OCVaX，谨以此小游戏纪念四胞胎震撼防晒妆造（bushi）。

---

## 游戏玩法

- 棋盘大小：8 × 8
- 图案数量：4 种普通图案 + 1 种五连合成的特殊图案
- 计分规则：
  - 3 消：+100 分  
  - 4 消：+200 分  
  - 5 消：+300 分，并在中间生成一个 **特殊方块**
- 特殊方块效果：
  - 和任意普通方块交换时，会**清除全盘所有相同图案**，并获得额外加分
- 彩蛋？
  - 达到若干特定分数时，会触发隐藏语音彩蛋，一共三个，快来找找吧！
- 特殊结局？
  - 当分数 **≥ ???** 时，会弹出特别提示： ？？？，并结束本局，在这就不剧透了！
- 当棋盘上**没有任何可行交换且能三消的步数**时，本局游戏结束（感觉很难彻底玩死，至少我没玩死过）

---

## 🕹 操作说明

- **点击两个相邻方块**即可交换
- **右上角「暂停」按钮**：
  - 暂停 / 继续游戏
  - 重新开始本局（分数清零，重新生成棋盘）
  - 音效开关设置
- 底部显示：
  - `Score`：本局当前分数
  - `Best`：历史最高分

## 技术栈

- **Phaser 3**：游戏主框架
- **原生 HTML / CSS / JavaScript**
- **LocalStorage**：本地记录最高分
- 自适应：
  - 使用 `Phaser.Scale.FIT + autoCenter` 让游戏在 PC / 手机上自动缩放并居中
